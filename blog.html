Important FAQ

What is the Document Object Model(DOM)?

The DOM (Document Object Model) in JavaScript is a programming interface that represents an HTML document as a structured tree of nodes. It allows developers to dynamically access, modify, add, or delete elements, attributes, and content within the document.

How do you Select an Element from the DOM?

Elements can be selected in different ways based on criterias:
1. getElementById() - Selects an element by its ID.
2. getElementsByClassName() - Selects elements by class name.
3. getElementsByTagName() - Selects elements by tag name.
4. querySelector() - Selects the first element matching a CSS selector.
5. querySelectorAll() - Selects all elements matching a CSS selector.

What is Event Delegation in the context of DOM and why it is useful?

Event delegation allows a parent element to manage events from its child elements by using a single event listener.
It's useful because it improves performance, reduces memory usage, and simplifies code by attaching fewer event listeners.>

How do you manipulate an elements' styles and attributes using the DOM?

You can manipulate an element's styles and attributes using the DOM by modifying the style property directly or by adding/removing CSS classes with classList. For attributes, use setAttribute() to change values, removeAttribute() to delete them, and getAttribute() to retrieve the current value of an attribute.
